- 自定义标题栏、窗口适应性调整
- 改善状态变化管理
- 字幕增加右键菜单，右击字幕单词可以加入单词本
- 导入单词方式变为粘贴文本
- 点击字幕单词自动翻译
- 增加播放历史，最多保存 10 条记录，用于支持“切换上个单词”
- 增加播放器快捷键

  - 点击视频区域才能激活快捷键
    | 快捷键 | 功能 |
    | ---- | ---- |
    | a | 单词生疏度 -100 |
    | g | 单词生疏度 +100 |
    | f | 视频最大化切换 |
    | 空格键 | 暂停/播放 |
    | j、ArrowDown | 下个单词 |
    | k、ArrowUp | 上个单词 |
    | h、ArrowLeft | 当前单词的上一视频 |
    | l、ArrowRight | 当前单词的下一视频 |
    | l | 当前单词的下一视频 |
    | l | 当前单词的下一视频 |

- 字幕字体大小自适应优化
- 暂停播放时视频区域显示播放按钮
